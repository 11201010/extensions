[同文安卓輸入法平臺](https://github.com/osfans/trime)（以下簡稱trime）的`配置`有`使用者資料夾`（可自定義路徑），負責儲存輸入法的方案檔及字典檔、虛擬鍵盤配置檔。

安裝[同文安卓輸入法平臺](https://github.com/osfans/trime)後，先下載RIME的初始化方案至`使用者資料夾`：`build`資料夾、`default.custom.yaml`。

重新佈署後，確定可顯示後，再下載[倉頡五代補完計劃](https://github.com/Jackchows/Cangjie5)的方案`cangjie5.schema.yaml`、字典檔`cangjie5.dict.yaml`至`使用者資料夾`。

最後，再下載個人改寫的虛擬鍵盤配置檔`虛舟.trime.yaml`至`使用者資料夾`。重新佈署後，在`主題`中選取主題。
